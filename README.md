# Map Generator v2-refactor

A LLM-mediated map generator. Enter a map request, e.g. "world map with Australia and the U.S. labeled, color them green" > Claude reads the request > returns a JSON that's fed to D3 > renders an SVG map you can save and cute up in Illustrator + a packaged D3.js interactive map you can embed.
